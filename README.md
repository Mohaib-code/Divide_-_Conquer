## 📊 Algorithms Overview

### 🔹 Merge Sort

* Time Complexity: **O(n log n)**
* Stable sorting algorithm
* Uses extra space

### 🔹 Quick Sort

* Average Time: **O(n log n)**
* Worst Case: **O(n²)**
* In-place sorting

### 🔹 Rotated Array Search

* Modified Binary Search
* Time Complexity: **O(log n)**
